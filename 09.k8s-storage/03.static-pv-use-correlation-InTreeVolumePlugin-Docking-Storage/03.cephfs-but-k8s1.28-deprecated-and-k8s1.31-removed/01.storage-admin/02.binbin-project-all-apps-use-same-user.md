## 1.创建client.binbinfs用户,并做相应的授权
```
##创建client.binbinfs用户,对mon有只读权限,对mds有读写权限,对cephfs-binbin-project-data有读写、执行权限

  ceph auth add client.binbinfs  mon 'allow r'  mds 'allow rw'  osd 'allow rwx pool=cephfs-binbin-project-data'

##导出client.binbinfs用户的key到本机 PS：后面要给到k8s管理员

  ceph auth print-key client.binbinfs   # 屏幕上打印
  ceph auth print-key client.binbinfs -o /tmp/ceph.client.binbinfs.secret
  ls -l /tmp/ceph.client.binbinfs.secret

## 其key为
  admin@ceph-mon01:~$ cat /tmp/ceph.client.binbinfs.secret
  AQAfO5BoC3wDLRAAO5RVDeuAZJWYE9ekZ3kOEg==
```
