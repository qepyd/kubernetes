# kubernetes的dashboard 
```
01:kubernetes dashboard 即 kubernetes集群通用的WEB UI。
02:kubernetes dashboard 可通过不同的软件来实现，例如:
   kubernetes组织的：
        可称为：kubernetes-dashboard
        代码库：https://github.com/kubernetes/dashboard
   kubeSphere组织的：
	可称为: kubeSphere-dashboard
        代码库：https://github.com/kubesphere/monitoring-dashboard
```
